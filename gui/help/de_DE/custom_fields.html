Benutzerdefinierte Felder

Die Informationen wurde aus dem Benutzerhandbuch des Mantis Bug-Tracking-Systems uebernommen und angepasst.

Beschreibung

Im Folgenden einige Hinweise zur Implementation von benutzerdefininierten Feldern:
  • Benutzerdefinierte Felder sind systemweit definiert.
  • Benutzerdefinierte Felder werden an einen bestimmten Elementtyp gehaengt (z.B. Testsuite, Testfall)
  • Benutzerdefinierte Felder koennen in verschiedenen Testprojekten verwendet werden.
  • Die Reihenfolge der benutzerdefinierten Feldern kann von Projekt zu Projekt abweichen.
  • Benutzerdefinierte Felder koennen fuer bestimmte Projekte auf inaktiv geschaltet werden.
  • Es gibt keine Begrenzung fuer die Anzahl der benutzerdefinierten Felder.

Die Definition eines benutzerdefinierten Feldes enthaelt folgende logische Attribute:
  • Name des Feldes
  • Name der Beschriftung (z.B.: Dies wird als Wert fuer die API-Funktion lang_get verwendet, oder einfach direkt angezeigt, wenn der Wert nicht in der Sprachendatei gefunden wird)
  • Typ des Feldes (String, Ganzzahlig, Fliesskommazahl, Aufzaehlung, E-Mail-Adresse)
  • Die moeglichen Werte der Aufzaehlung (z.B.: ROT|GELB|BLAU), die bei normalen Listen, bei Multi-Selektions-Listen und bei Auswahlboxen verwendet werden koennen.
    Benutze das Pipe-Symbol ('|') um die einzelnen Aufzaehlungswerte zu trennen. Einer der Werte kann dabei ein leerer String sein.
  • Default-Wert. (NOT IMPLEMENTED YET)
  • Minimum/Maximum-Laenge fuer den Feldwert (0 fuer keine Begrenzung). (NOT IMPLEMENTED YET)
  • Ein regulaerer Ausdruck, um die Eingabe zu validieren. (benutzer ereg() -Syntax). (NOT IMPLEMENTED YET)
  • Alle benutzerdefinierten Felder werden derzeit in einem VARCHAR(255)-Feld in der Datenbank gespeichert.
  • Anzeige waehrend der Test-Spezifikation.
  • Editieren waehrend der Test-Spezifikation. Der Benutzer kann den Wert waehrend des Designs der Test-Spezifikation aendern.
  • Anzeige waehrend der Testausfuehrung.
  • Editieren waehrend der Testausfuehrung. Der Benutzer kann den Wert waehrend der Test-Ausfuehrung aendern.
  • Verfuegbar fuer. Der Benutzer waehlt aus, zu welcher Elementart das Feld gehoert.
