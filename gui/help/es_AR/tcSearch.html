Test Case Search Page

Purpose:

Navigation according to keywords and/or searched strings. The search is not case sensitive. Result include just test cases from actual product (a difference against version 1.0.4)

To search:

  1. Write searched string to an appropriate box. Left blank unused fields in form.
  2. Choose required keyword or left value 'Not applied'.
  3. Click the Search button.
  4. All fulfilled test cases are shown. You can modify test cases via 'Title' link.
