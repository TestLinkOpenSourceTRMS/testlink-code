Assegna Utenti a Test Plan

Purpose:

Utenti con le autorizzazioni necessarie possono definire l'elenco degli utenti autorizzati per ogni Test Plan. Solo gli utenti di questo elenco possono modificare il Test Plan.

Operazione

  1. Scelta tra utenti e test plan.
  2. Clicking on a user will allow you to select all of the Test Plan that the user is able to see.
  3. Selecting a test Plan allows you to select all of the users that are a part of that Test Plan.
