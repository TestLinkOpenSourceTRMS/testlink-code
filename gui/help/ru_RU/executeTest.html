Прогнать тесты

Цель:

Этот раздел предназначен для прогона тестов. Пользователь может задать результат прогона теста для каждой версии (сборки). Подробно о выборе параметров отбора тестов для прогона см. Помощь .

Что делать:

  1. Пользователь должен определить тестируемую версию (сборку) и набор тестов в рамках Плана тестирования.
  2. Выбрать версию (сборку) Продукта из выпадающего меню и нажать кнопку "Обновить меню" на навигационной панели.
  3. Выбрать компонент для просмотра его категорий и всех тестовых примеров, которые к ним относятся.
  4. Выбор категории покажет тестовые примеры только этой категории.
  5. Заполнить данные о результатах тестирования для каждого тестового примера, а также записать информацию о любых возникших проблемах, замечаниях и багах программы.

Примечание: Если в плане тестирования не определено ни одного теста, то в окне прогона будет доступен только значок информации.
