Tester zu einem Testplan hinzufuegen

Ziel:

Diese Seite erlaubt es dem Benutzer einem Testplan eine Anzahl berechtiger Benutzer hinzuzufuegen. Diese Benutzer duerfen dann Daten innerhalb des Testplans bearbeiten.

Anleitung:

  1. Waehle entweder aus die Rechte aus Sicht der Benutzer oder aus Sicht der Projekte zu bearbeiten.
  2. Wenn man einen Benutzer auswaehlt, sieht man alle Testplaene, die der Benutzer sehen darf.
  3. Wenn man einen Testplan auswaehlt, kann man alle Benutzer auswaehlen, die diesem Testplan zugeordnet sind.
