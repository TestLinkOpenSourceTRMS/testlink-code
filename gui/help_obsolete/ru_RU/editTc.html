Редактирование тестов

Назначение:

Раздел "Описание тестов" – это место, где пользователь может просмотреть и отредактировать компоненты, категории тестов,а также описания тестовых примеров, относящихся к заданному Продукту. Пользователь не может просмотреть различные версии теста – база данных содержит только последнюю (действующую) версию. Рекомендуется распечатывать описания тестов в ходе тестирования.

Что делать:

  1. Выберите название продукта в из выпадающего меню, расположенного в правом верхнем углу экрана.
  2. Создайте новый компонент, категорию или тест для этого продукта.
  3. Используя древовидную структуру панели навигации слева, отредактируйте описания тестовых примеров.
  4. Когда тест будет готов, включите его в План тестирования.

Все тестовые примеры в TestLink’е распределяются по Компонентам и Категориям. Они имеют несколько полей для описания характеристик входящих в них групп тестов. Эта информация может быть распечатана вместе с описанием самого теста.
